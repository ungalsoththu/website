# Roadmap: UngalSoththu

## Overview

Build a bilingual (Tamil + English) static website for urban transit accountability in Chennai and Tamil Nadu. The journey progresses from foundational setup through design systems, content and data infrastructure, content pages, interactive mini apps, and finally launch with accessibility validation.

## Domain Expertise

None
